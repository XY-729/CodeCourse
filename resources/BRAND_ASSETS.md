# CodeCourse Brand Asset Lock

The approved CodeCourse mark is the charcoal-to-green interlocking zigzag selected on 2026-07-16.

Canonical desktop and web assets:

- `resources/icon.png`: high-resolution raster source.
- `resources/icon.ico`: Windows multi-size application icon.
- `resources/icon.svg`: editable vector counterpart.
- `frontend/public/logo.png`: in-app and touch icon.
- `frontend/public/favicon.ico`: browser tab icon.

Do not replace, regenerate, recolor, or revert these assets as part of routine feature work, packaging, release automation, or mobile development. A logo change requires an explicit user request that names the logo change.

Android launcher assets are intentionally outside this lock and were not changed in this update.
